# BiliAva 🖼️

> B站用户头像原图查看工具 | User Avatar Original Image Viewer for Bilibili

---

## 🚀 核心功能

### 🖼️ 头像查看优化
- 一键查看用户空间头像原图
- 快速打开用户卡片头像原图
